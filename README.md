# Tugas Besar Pemrograman Web I

Project ini merupakan **Tugas Besar Mata Kuliah Pemrograman Web I** yang bertujuan untuk menerapkan konsep dasar pengembangan website statis menggunakan HTML, CSS, dan JavaScript.

Website ini mengangkat tema Artikel game Hollow Knight, dengan menampilkan informasi singkat mengenai dunia game, konten DLC, serta elemen multimedia dan interaksi sederhana.

Tujuan Project
- Memahami struktur dasar HTML
- Menerapkan styling menggunakan CSS
- Menggunakan JavaScript untuk interaksi sederhana
- Mengelola file dan asset website secara terstruktur
- Mempublikasikan website menggunakan GitHub Pages

Fitur Website
- Halaman utama berisi deskripsi Hollow Knight
- Navigasi antar halaman DLC (Dream, Grim Troupe, Godmaster)
- Tampilan gambar ilustrasi game
- Pemutar soundtrack latar
- Form komentar sederhana
- Tombol navigasi kembali ke atas halaman
